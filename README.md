post.css is a css library for markdown document. 

## Install

### Using the `<link>` tag

```html
<!-- cdn -->
<link rel="stylesheet" href="https://unpkg.com/post-style/lib/post.min.css">
```

### Using in Sass/Scss

```
npm i -D post-style
```

```scss
@import "~post-style";
```
